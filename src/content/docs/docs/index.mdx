Kito is a high-performance, type-safe TypeScript web framework powered by a Rust core. It combines the developer experience of TypeScript with the blazing speed and safety of Rust.

## Quick Example

```typescript
import { server, schema, t } from "kitojs";

const app = server();

const userSchema = schema({
  body: t.object({
    name: t.str().min(1),
    email: t.str().email()
  })
});

app.post("/users", [userSchema], ({ req, res }) => {
  const { name, email } = req.body;
  res.json({ name, email });
});

app.listen(3000);
```

## Key Features

#### Extreme Performance

Kito consistently outperforms Node.js frameworks thanks to its Rust core:

<img src="https://raw.githubusercontent.com/kitojs/kito/main/bench/results/charts/result-node.png" />

#### Type-Safe by Default

Every request is validated and typed:

```typescript
// TypeScript knows exact types
ctx.req.body.name;   // string
ctx.req.body.email;  // string
ctx.req.params.id;   // string (if validated)
```

#### Extensible

Extend the context with custom services:

```typescript
const app = server().extend<{ db: Database }>(ctx => {
  ctx.db = database;
});

app.get("/users", async ({ db, res }) => {
  const users = await db.query("SELECT * FROM users");
  res.json(users);
});
```

#### Cross-Runtime

Works seamlessly on:

- Node.js (v18+)
- Deno (v1.30+)
- Bun (v1.0+)

Same code, same API, everywhere.
