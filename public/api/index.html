    Minimal Example

    import { server } from "kitojs";
    const app = server();

    app.get("/", ({ res }) => {
      res.send("hello world!");
    });

    app.listen(3000);
    Fluent style

    Kito also supports fluent style. You can chain all methods. See the examples here.

    import { server } from "kitojs";

    server()
      .get("/", ({ res }) => res.send("hello world!"))
      .listen(3000);
